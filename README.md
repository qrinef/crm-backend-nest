## Live demo

This demo app is deployed at [crm-frontend-nuxt.qrinef.ru](https://crm-frontend-nuxt.qrinef.ru)

## Environment variables
```bash
- DB_DIALECT
- DB_HOST
- DB_PORT
- DB_USERNAME
- DB_PASSWORD
- DB_DATABASE
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
